# As Is

**What you see is what you get. No refunds. No crying.**

As Is is a raw, chaotic, no-bullshit marketplace where everything is sold **strictly As Is**.

## Core Philosophy
- Everything is sold As Is, Where Is, With All Faults
- Minimal platform interference
- Buyers and sellers are expected to act like adults
- Fast, cheap, and transparent

**Only 3 Exceptions** where the platform will step in:
1. Package never arrives according to tracking
2. Package arrives clearly empty
3. Buyer receives a completely different category of item than listed

## Key Features
- Rage Buy & All Mine (Live only)
- Juice boosting ($0.99)
- Live selling with chat + games
- 4 Seller Levels with increasing benefits
- Promotional Spotlight (paid ads)

## Design
- Dark Mode by default
- Themes: Exotic Dark + Explosive Dark available
- Tone: Blunt, direct, slightly vulgar
