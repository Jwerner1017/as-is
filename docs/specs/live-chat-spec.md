# Live Chat Specification

## Features
- Real-time chat during live streams
- 3 moderation levels that affect what gets filtered:
  - Relaxed (default)
  - Medium
  - Strict
- Users can send tips directly from chat
- Chat should feel fast and chaotic (matching the platform tone)

## Message Types
- Normal chat messages
- Tip messages (with amount and optional public/private setting)
- System messages (user joined, Rage Buy triggered, etc.)
- Moderation actions (message deleted, user timed out)

## Rules
- No doxxing or sharing personal information
- No off-platform deal attempts
- Harassment and threats = instant ban from chat + possible account action
- Talking shit and roasting is generally allowed (especially in Relaxed mode)
