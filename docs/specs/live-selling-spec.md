# Live Selling Specification

## Core Rules
- All live sales are still governed by the normal As Is Policy (no extra buyer protection)
- Sellers must show the physical item on camera during the stream
- No nudity or sexually explicit content allowed (including in the background)
- Sellers can choose from 3 chat moderation levels:
  - Relaxed (default)
  - Medium
  - Strict

## Allowed Features
- Live chat with moderation
- Tips (seller keeps 100%)
- Live games and contests (Giveaways, Mystery Boxes, Spin the Wheel, Pick a Number, etc.)
- Rage Buy + All Mine (only available during live streams)

## Requirements for Sellers
- Must be on camera and show the item clearly
- Must accurately represent the item being sold
- All sales during live are final (As Is rules apply)

## Moderation Levels
- **Relaxed**: Maximum freedom (trash talk allowed, within reason)
- **Medium**: Moderate filtering of extreme language
- **Strict**: Clean chat only
